Descripción detallada

Funciones de formateo avanzado para int128_t.

Este archivo proporciona funciones que simulan el comportamiento de los manipuladores iostream para int128_t, ya que el operador<< básico no los respeta automáticamente. Incluye manejo especializado para números negativos y signo.
