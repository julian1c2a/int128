Descripción detallada

Especializaciones de funciones de rango de std para int128_t.

Este archivo proporciona especializaciones de las funciones de rango estándar (std::iota, std::accumulate, etc.) para trabajar con int128_t. Incluye manejo especializado para enteros con signo y overflow.
