Descripción detallada

Forward declarations for uint128_t type traits specializations.

⚠️ CRITICAL: This header MUST be included BEFORE <type_traits> is included for the first time. Otherwise, the specializations will not be recognized.

This header only contains forward declarations and specializations for std::type_traits. It does NOT include uint128_t.hpp to avoid circular dependencies and ensure traits are available when type_traits is first included.
